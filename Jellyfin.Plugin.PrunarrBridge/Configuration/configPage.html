Prunarr Bridge Configuration

Configure the Prunarr Bridge plugin to manage "Leaving Soon" media items via symlinks.

Note: This plugin uses Jellyfin's built-in authentication. Use your Jellyfin API key when calling the endpoints.

Directory where symlinks for "Leaving Soon" items will be created. Default: /var/lib/jellyfin/leaving-soon
Name of the virtual folder/library for "Leaving Soon" items. Default: Leaving Soon
Automatically create the "Leaving Soon" virtual folder when Jellyfin starts if it doesn't exist.
