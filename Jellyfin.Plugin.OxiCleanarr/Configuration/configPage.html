OxiCleanarr Bridge

Billy Mays here! This plugin is like OxiClean for your media library - it cleans up symlinks with the power of oxygen action!

The OxiCleanarr Bridge plugin manages symlinks via HTTP API. It's stateless and configuration-free!

API Endpoints

  • POST /api/oxicleanarr/symlinks/add - Create symlinks
  • POST /api/oxicleanarr/symlinks/remove - Remove symlinks
  • GET /api/oxicleanarr/symlinks/list - List all symlinks
  • GET /api/oxicleanarr/status - Get plugin status

Authentication

This plugin uses Jellyfin's built-in authentication. Use your Jellyfin API key when calling the endpoints.
